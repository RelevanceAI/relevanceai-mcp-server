# MCP Proxy POC

## Setup

1. clone repo
1. `npm i`
1. `cp .env .env.example` & add your relevance auth token and region
1. Copy path to the `run.sh` file: ![](/static/run-sh.png)
1. paste in MCP thing
   ![](/static/mcp-thing.png)
   then click "Add"
   ![](/static/mcp-ready.png)
1. profit!!
   ![](/static/google-oranges.png)
